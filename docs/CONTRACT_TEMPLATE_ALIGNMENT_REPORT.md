# Contract Template Alignment Report

The generated Cor House rental contract has been aligned closer to the real Fastighets Ab Cor-huset contract structure without copying personal data from the signed sample.

Implemented alignment:

- Landlord header: Fastighets Ab Cor-huset.
- ASK role: Kontaktperson / administratör.
- Formal section order and plain document style.
- Cor-huset rental object labels: Festsal, Kabinett, Bastu och kabinett, Kök (med kärl).
- Bokningsreferens and Avtalsdatum are present but not dominant.
- Avtalsperiod uses DD.MM.YYYY and 24-hour time.
- Hyra uses the stored booking price breakdown.
- Swedish terms include cleaning, damage, inspection, fire alarm, invoicing, and two-copy language.
- Door code appears only under Tillträde / dörrkod.
- Signature section uses Helsinki date blanks and landlord/tenant representative lines.

Configurable defaults are provided for Fastighets Ab Cor-huset company details, bank details, and ASK contact details.
